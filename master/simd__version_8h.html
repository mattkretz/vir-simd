◆ VIR_SIMD_VERSION

#define VIR_SIMD_VERSION   0x0'04'bd

The current version as a single integer.

The least significant 8 bits represent the patchlevel:

  • release >= 0x00 and even
  • development >= 0x00 and odd
  • alpha >= 0xbe
  • beta >= 0xc8
